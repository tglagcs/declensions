# Склонение ФИО

Минималистичный веб-инструмент для автоматического склонения русских фамилий, имён и отчеств по всем шести падежам с определением рода.

**[→ Открыть сайт](https://tglagcs.github.io/declensions/)**

---

## Возможности

- **6 падежей** — Именительный, Родительный, Дательный, Винительный, Творительный, Предложный
- **Автоопределение рода** — по отчеству (мужской / женский)
- **Live-режим** — результат обновляется при каждом вводе символа после первого склонения
- **Копирование в один клик** — с toast-уведомлением
- **Адаптивный дизайн** — работает на мобильных устройствах

## Технологии

Чистый HTML / CSS / JavaScript — без фреймворков и зависимостей.

## Запуск локально

```bash
git clone https://github.com/tglagcs/declensions.git
cd declensions
# Открыть index.html в браузере
```
